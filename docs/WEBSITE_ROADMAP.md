# Website Roadmap

## Short term

- Publish the image inverse problems repository, figures, and mini report when ready.
- Add a real public email or GitHub profile only when you want them visible.
- Replace broad course labels with exact course names from official records.

## Medium term

- Add a writing content collection after the first complete reflection or learning note exists.
- Add verified project images and accessible captions.
- Expand the statistical analysis project around a concrete public dataset.

## Later

- Add bilingual project reflections where Chinese explanation improves clarity.
- Consider RSS only after Writing contains enough real posts.
- Review accessibility and performance after adding media.

## Not yet

- No CMS, comments, search, or analytics until there is a clear need.
- No empty social links, invented achievements, or decorative project pages without real content.
