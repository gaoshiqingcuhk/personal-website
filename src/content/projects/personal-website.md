## Overview

This website is a small meta project about organizing and presenting ongoing work. It began as an academic portfolio and is gradually becoming a concise place for selected projects, CV information, and learning notes.

## What I Built

- A static Astro and TypeScript site deployed to a custom domain
- Markdown-based project pages
- Shared profile and CV data structures
- A browser-friendly CV and a print-ready PDF workflow
- Responsive layouts without a heavy UI framework

## Reflection

The main design challenge is balance. The site should remain credible for academic and project contexts while still feeling like a maintained personal space.
