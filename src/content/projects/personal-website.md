## Overview / 项目概览

This website is a small meta project about how I organize and present ongoing work. It began as an academic portfolio and is gradually becoming a broader bilingual space for projects, writing, CV, and personal interests.

这个网站本身也是一个持续迭代的项目：它记录我如何组织内容、调整表达、做取舍，并把零散学习成果整理成可以长期维护的公开空间。

## What I built / 做了什么

- A static Astro and TypeScript site deployed to a custom domain
- Markdown-based project pages
- Shared profile and CV data structures
- A browser-friendly CV and a print-ready PDF workflow
- Responsive bilingual layouts without a heavy UI framework

## Reflection / 复盘

The main design challenge is balance. The site should remain credible for academic and project contexts, but it should not feel like an application document stretched into a website. The current structure leaves room for technical work, writing, and small personal explorations without pretending that every section is already complete.

## Next steps / 下一步

- Add real writing as it is completed
- Improve project visuals when verifiable figures are available
- Continue refining accessibility and mobile reading
